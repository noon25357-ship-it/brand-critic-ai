# Brand Critic AI

Evidence-based AI skill for auditing:

- Logos
- Landing pages
- SaaS dashboards
- App interfaces
- UX
- Copywriting
- Conversion

## Installation

Copy the `SKILL.md` file into:
```text
.claude/skills/brand-critic-ai/SKILL.md
```

## Usage

Open Claude Code, then send a logo, website, landing page, app screenshot, or ad creative and write:

```text
حلل هذا البراند
```

The skill starts immediately and adapts its analysis to the material provided.

## What It Produces

- Brand Scorecard
- UX and conversion review
- Biggest problems
- Conversion killers
- Quick wins
- Redesign plan
- AI prompts for execution
- Final verdict
- Shareable X roast
